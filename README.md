متجر الأحذية الإلكتروني
رجالي | حريمي | أطفال
================================

نبذة عن المشروع
---------------
متجر أحذية إلكتروني حديث يوفّر تجربة تسوّق تفاعلية لعرض وبيع الأحذية
الرجالية والنسائية وأحذية الأطفال، بتصميم عصري يدعم العرض ثلاثي الأبعاد (3D)
ويعمل بكفاءة على الهواتف المحمولة وأجهزة سطح المكتب.

يهدف المشروع إلى تسهيل عملية اختيار الأحذية عبر عرض بصري واقعي،
وربط المتجر بمنصة إنستجرام لزيادة الوصول والتفاعل مع العملاء.


أهداف المتجر
------------
- توفير تجربة تسوّق سهلة وسلسة.
- عرض الأحذية بتقنية ثلاثية الأبعاد.
- دعم فئات: رجالي – حريمي – أطفال.
- تسهيل عمليات الدفع الإلكتروني.
- تعزيز المبيعات عبر وسائل التواصل الاجتماعي.


أقسام المتجر
------------
- أحذية رجالي
- أحذية حريمي
- أحذية أطفال
- أحدث الإضافات
- الأكثر مبيعًا
- البحث والتصفية حسب المقاس والسعر والنوع


التصميم وواجهة المستخدم
------------------------
- تصميم ثلاثي الأبعاد (3D).
- ألوان أساسية: بني جملي.
- تصميم متجاوب مع:
  * الهواتف المحمولة
  * أجهزة الكمبيوتر
- واجهة استخدام سهلة وسريعة.


طرق الدفع
----------
- الدفع عبر محفظة شركة جوال.
- متوافق مع قطاع غزة.
- آمن وسريع.
- قابل للتطوير مستقبلًا لإضافة وسائل دفع أخرى.


التكامل مع وسائل التواصل
-------------------------
- ربط مباشر مع Instagram.
- مشاركة المنتجات على إنستجرام.
- تحويل المستخدمين من إنستجرام إلى صفحة المنتج.


التقنيات المستخدمة
-------------------
- HTML
- CSS
- JavaScript
- تقنيات العرض ثلاثي الأبعاد (مثل Three.js أو Model Viewer)


طريقة التشغيل
--------------
1. تحميل ملفات المشروع.
2. رفع الملفات على استضافة أو خادم محلي.
3. إعداد بيانات الدفع (محفظة جوال).
4. ربط حساب إنستجرام.
5. تشغيل المتجر وبدء البيع.


التطوير المستقبلي
------------------
- نظام حسابات المستخدمين.
- تتبع الطلبات.
- نظام تقييم ومراجعات.
- لوحة تحكم للإدارة.
- دعم لغات متعددة.


الترخيص
--------
هذا المشروع مخصص للاستخدام التجاري ويمكن تعديله وتطويره حسب الحاجة.
